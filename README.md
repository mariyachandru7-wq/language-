Language Translator Chatbot

📌 Project Description

This project is a simple Language Translator Chatbot built using Python. It allows users to enter text and get it translated into Tamil.

🚀 Features

- Automatic language detection
- Translates text into Tamil
- Simple command-line interface

🛠️ Technologies Used

- Python
- deep-translator library

▶️ How to Run

1. Install the required library:
   pip install deep-translator

2. Run the program:
   python translator.py

- translator.py → Main Python script
